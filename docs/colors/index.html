Colors

Complete system

Our color system includes of 7 hues (including grey), each consisting of 10 shades. The scale goes from the numbers 10-100. Each color in the scale is named according to the scheme {color}-{number}, such as orange-50.

You can read more about our brand colors (building upon this system) in "Brand".

Wide gamut colors

We make use of wide gamut colors with the Display P3 color space on supported devices and browsers. These colours look more vibrant and saturated. All color variables in the CSS are automatically in the Display P3 space if the browser supports it.
